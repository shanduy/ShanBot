## Instalar El Bot
Pasos para instalar el bot en termux

```bash
> pkg install git
> pkg install ffmpeg
> pkg install nodejs
> pkg install wget
> git clone https://github.com/shanduy/ShanBot
> cd ShanBot
> bash install.sh
> npm start
> Y escaneas el QR con tu Whatsapp
```
